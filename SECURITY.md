# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Fourth Coast Web LLC takes the security of **Image Metadata Generator** seriously. If you have discovered a security vulnerability, we kindly ask that you do **not** disclose it publicly until we have had a chance to fix it.

### How to Report

Please email us at [info@fourthcoastweb.com](mailto:info@fourthcoastweb.com) with the following details:

- **Type of vulnerability:** (e.g., XSS, CSRF, SQL Injection, etc.)
- **Full description:** Detailed steps to reproduce the vulnerability.
- **Proof of Concept:** Code or instructions that demonstrate the vulnerability.

### Our Response

We will acknowledge your report within 48 hours and provide an estimated timeline for a fix. We appreciate your responsible disclosure and will credit you for your discovery (unless you wish to remain anonymous) once the vulnerability has been patched.

Thank you for helping keep our plugin safe!
