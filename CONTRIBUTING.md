# Contributing to Fourth Coast Web's Image Metadata Generator

Thank you for your interest in contributing to **Image Metadata Generator**! We welcome contributions from the community to help improve this plugin.

## How to Contribute

### Reporting Bugs

If you find a bug, please create a new issue in the issue tracker. Be sure to include:

- A clear, descriptive title.
- Steps to reproduce the issue.
- The version of the plugin and WordPress you are using.
- Any relevant error logs or screenshots.

### Suggesting Enhancements

We love ideas for new features! Please submit an issue with the "enhancement" label. Describe the feature you'd like to see and why it would be useful.

### Pull Requests

1.  **Fork** the repository on GitHub.
2.  **Clone** your fork to your local machine.
3.  **Create a branch** for your feature or bugfix (`git checkout -b feature/amazing-feature`).
4.  **Make your changes**. Ensure you follow the coding standards (see below).
5.  **Commit** your changes with clear, descriptive commit messages.
6.  **Push** to your branch (`git push origin feature/amazing-feature`).
7.  **Open a Pull Request** against the `main` branch of the original repository.

## Coding Standards

- **PHP:** We follow the [WordPress PHP Coding Standards](https://developer.wordpress.org/coding-standards/wordpress-coding-standards/php/).
- **JS/CSS:** Follow standard WordPress best practices.
- **Documentation:** Please ensure any new features are documented in the code and potentially in the README.

## Development Environment

To set up a local development environment:

1.  Ensure you have a local WordPress instance running.
2.  Clone this repository into `wp-content/plugins/`.
3.  Run `composer install` (only if you are adding dev dependencies).
4.  Activate the plugin.

Thank you for helping make this plugin better!
